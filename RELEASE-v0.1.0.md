# Release v0.1.0

## English

Initial public release of reusable Feishu/Lark office automation templates built on top of `lark-cli`.

### Highlights

- added reusable `lark` command templates for workspace, message, calendar, document, and task workflows
- added bilingual `office:workspace` routing template
- added Chinese office command templates for workspace, message, calendar, document, and task scenarios
- added open-source repository essentials including license, contributing guide, changelog, and bilingual readmes

### Recommended Use Cases

- daily Feishu assistant workflows
- structured office automation prompts
- reusable command packs for internal tooling
- bilingual command entry points for Chinese-speaking teams

## 中文

这是基于 `lark-cli` 的飞书/Lark 办公自动化模板首个公开版本。

### 版本亮点

- 增加了 `lark` 基础命令模板，覆盖 workspace、message、calendar、document、task 场景
- 增加了双语统一路由模板 `office:workspace`
- 增加了中文办公子命令模板，覆盖 workspace、message、calendar、document、task 场景
- 补齐了开源仓库基础文件，包括许可证、贡献指南、变更日志和中英文 README

### 适用场景

- 日常飞书办公助手
- 结构化办公自动化提示词
- 内部工具可复用命令模板包
- 面向中文团队的双语命令入口
